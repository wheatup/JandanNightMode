## 离线安装方法

* 下载[JandanNightMode1.2.4.crx](https://github.com/wheatup/JandanNightMode/raw/master/JandanNightMode1.2.4.crx)文件
* 打开chrome
* 在地址栏中输入
```
chrome://extensions/
```
* 回车
* 将下载好的crx文件拖进浏览器中
